Rubric for final project report (20pts) 
Wednesday April 29 – Final project reports due
Abstract (2pt)
Provides a concise summary of the project and effectively highlights key insights/findings.
Introduction/Motivation (2pt)
Is the problem described clearly?
Why is it important?
What is your goal
Related work (2 pt)
How have people tackled this problem before
How is your method different from prior work?
Methodology (6 pt)
The challenge is stated clearly: why is it not easy to come up with the method? What is the novelty (if any)?
The method is described clearly (preprocessing steps, the model etc.).
Key findings and insights of the project are effectively demonstrated and discussed. 
Experimental results (6 pt)
Settings should be meaningful and clearly defined: (i.e., baselines,  performance metrics, ablations, applications, etc.).
Readers can understand the purpose of each experiment and how it relates to the research problem the project tackles. 
Can the experiments or demo be reproduced from the information given in the report? (including code-base)
Clarity and format (2 pt)
Include a section that lists the individual contributions of each team member
The report should be at 5-6 pages in length (10pt font minimum, please don’t make it too dense)
Submit in PDF format
